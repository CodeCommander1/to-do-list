# TO-DO LIST in C++

A terminal-based To-Do List application written in C++.

## Features

- ✅ Add Task
- 📄 View Tasks
- ✔️ Mark Task as Completed
- ❌ Remove Task
- 🚪 Exit Program

## How to Run

```bash
g++ todo.cpp -o todo
./todo   # or todo.exe on Windows
```
